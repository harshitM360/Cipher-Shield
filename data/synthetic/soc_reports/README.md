# SOC Reports

This folder stores the final analyst-facing SOC reports generated in Phase 8.

Naming pattern:
- `<scenario_name>_soc_report.json`

Each report combines data from these earlier pipeline stages:
- `scenarios/`
- `generated/`
- `feature_summaries/`
- `rule_hits/`
- `risk_assessments/`
- `mitre_mappings/`

These reports are the explainable output layer of the ransomware detection copilot.
